##Version##
All data were analyzed in RStudio (v4. 3. 1). 

##Data##
These data were generated from house finches (n=150) which were either inoculated with one of two doses of 
Mycoplasma gallisepticum (MG) or sterile control, allowed to recover, then rechallenged with one of five
secondary doses. Blood was collected from birds 8 days prior to inoculation as well as 14 and 41 days post-inoculation
with MG, from which antibody levels were quantified using an anti-MG ELISA kit. Pathogen loads and pathology
(eye score) were quantified across the course of the experiment.

##Replicability##
Reinfection_Augments_Heterogeneity.R contains all code for analyses and figures in the manuscript.

For a complete re-analysis including data cleaning, run dataCleaning_EEID1A.R to generate reinfection_response.rds for
Reinfection_Augments_Heterogeneity.R. The data cleaning script cleans up the raw data for analysis.

##Data Files##
1. EEID2021_master_data_20220503.csv
Description: Raw data file with data pathology, pathogen load, and susceptibility data from all experimental birds across all experimental days

2. EEID_1a_ELISA.csv
Description: Raw data file with antibody data from all experimental birds

3. reinfection_response.rds
Description: .rds file generated by merging 1. and 2. into working data frame for analysis

4. all_pseudosets.csv
Description: Data generated in Hawley et al., 2024, PLoS Pathogens used in CIs_Sus_2024.R to calculate CoV for Figure 4

5. deviance_params_noday41positives.csv
Description: Data generated in Hawley et al., 2024, PLoS Pathogens used in CIs_Sus_2024.R to calculate CoV for Figure 4

Link to original README.R for 4. and 5: https://github.com/klangwig/mg_dose_response_public/blob/main/README.md

##R Scripts##
1. dataCleaning_EEID1A.R
Description: Data cleaning and formatting script to generate working dataframe from raw data
Requires: EEID2021_master_data_20220503.csv, EEID_1a_ELISA.csv
Produces: reinfection_response.csv, reinfection_response.rds"

2. Variability_Functions.R
Description: Functions to calculate variability metrics used in Reinfection_Augments_Heterogeneity.R

3. CIs_Sus_2024.R
Description: Script used to calculate 95% CIs for CoV from gamma distributed susceptibility from Hawley et al., 2024
Requires all_pseudosets.csv, deviance_params_noday41positives.csv

4. Reinfection_Augments_Heterogeneity.R
Description: Script for all analyses and figures
Requires: reinfection_response.rds, Variability_Functions.R, CIs_Sus_4042.R
